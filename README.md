# HelloWorldAndroidApp

This is a minimal Android "Hello World" app. The project includes a simple `MainActivity` that displays a text view with the message "Hello World!".

## Installation

1. Clone this repository.
2. Build the project using Android Studio or the Android Gradle plugin.
3. Alternatively, download the prebuilt APK below and install it on your Pixel 8 Pro or any Android device:

[Download HelloWorldAndroidApp.apk](./HelloWorldAndroidApp.apk)

## Building from source

If you have the Android SDK installed, you can build the debug APK with:

```bash
gradle assembleDebug
```

The resulting APK will be located at `app/build/outputs/apk/debug/`.
